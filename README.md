# Schizophrenia Classification using ICA-FNC

## Overview

This project investigates functional network connectivity (FNC) differences between schizophrenia patients and healthy controls using resting-state fMRI data from the COBRE dataset.

## Methods

- Group ICA (FastICA)
- Dual regression
- Functional Network Connectivity (FNC)
- Statistical testing (t-test + FDR correction)
- Logistic regression classification

## Key Result

A significant reduction in connectivity between the Default Mode Network (DMN) and Somatomotor Network was found:

- t = 3.70  
- q = 0.016 (FDR corrected)  
- Cohen's d = 0.74
